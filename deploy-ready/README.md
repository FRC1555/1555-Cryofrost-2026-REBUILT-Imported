# Deploy-Ready Folder

This folder collects the files students are most likely to need when preparing to build and deploy the robot quickly.

## Important

- These are mirrored copies for fast access.
- The actual Gradle and WPILib build still uses the real files in the main repo tree.
- If you edit something here, copy the same change into the real source tree before building.

## Real Source Locations

- Java code: `src/main/java/frc/robot/`
- PathPlanner autos and paths: `src/main/deploy/pathplanner/`
- Vendor dependencies: `vendordeps/`
